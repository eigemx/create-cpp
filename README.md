# createcpp
`createcpp` is my personal tool to quickly create a C++ project with CMake, and my own setup for clang-format and clang-tidy.

## Installation
clone the repo: 
```
git clone https://github.com/eigemx/createcpp
```

install via pip:
```
cd createcpp
pip install .
```

## Usage
```
createcpp helloworld
```
